## 🎵 Music Store Data Analysis (SQL Project)
📌 Overview

This project focuses on analyzing a digital music store database using SQL. The goal is to extract meaningful insights about customer behavior, sales trends, and music preferences using structured queries.

### 📂 Dataset Description

The dataset simulates a music store and includes the following tables:

- Customers – Customer details (name, country, email)
- Invoices – Purchase transactions
- Invoice_Items – Details of each purchased track
- Tracks – Song information
- Albums – Album details
- Artists – Artist information
- Genres – Music categories
- Employees – Staff details

### 🎯 Objectives
- Analyze customer purchasing patterns
- Identify top-selling tracks, artists, and genres
- Determine revenue trends
- Understand geographic sales distribution

### 📊 Key SQL Concepts Used
- SELECT, WHERE, ORDER BY
- JOINS (INNER JOIN, LEFT JOIN, etc.)
- GROUP BY & HAVING
- Aggregate Functions (SUM, COUNT, AVG)
- Subqueries
- Window Functions (optional advanced queries)
